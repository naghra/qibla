# موقع بطاقة الوصول الرقمية التايلاندية TDAC

موقع ويب مشابه لخدمات المساعدة في تقديم بطاقة الوصول الرقمية التايلاندية (TDAC)، باللغة العربية واتجاه RTL.

## المميزات

- صفحة هبوط كاملة مع جميع الأقسام (معلومات، مميزات، خطوات، أسعار، دول، شهادات، أسئلة شائعة)
- نموذج تقديم متعدد الخطوات مع دعم عدة مسافرين
- ثلاث خطط أسعار (قياسي، سريع، فائق السرعة)
- تصميم متجاوب يعمل على الجوال والحاسوب
- واجهة عربية كاملة (RTL)

## التشغيل

```bash
npm install
npm run dev
```

افتح المتصفح على `http://localhost:3000`

## التشغيل على الهاتف (نفس شبكة الواي فاي)

1. على الحاسوب، شغّل المشروع:
   ```bash
   npm install
   npm run dev
   ```
2. اعرف عنوان IP المحلي للحاسوب:
   - **Windows:** `ipconfig` ثم ابحث عن `IPv4 Address`
   - **Mac:** `ipconfig getifaddr en0`
   - **Linux:** `hostname -I`
3. على هاتفك (متصل بنفس الواي فاي)، افتح المتصفح واكتب:
   ```
   http://عنوان-IP-الحاسوب:3000
   ```
   مثال: `http://192.168.1.15:3000`

> عند تشغيل `npm run dev` سيظهر في الطرفية سطر `Network:` يحتوي الرابط الجاهز للهاتف.

### إذا لم يفتح على الهاتف
- تأكد أن الحاسوب والهاتف على **نفس شبكة الواي فاي**
- عطّل VPN مؤقتاً على الحاسوب أو الهاتف
- اسمح للمنفذ `3000` في جدار الحماية (Firewall) على الحاسوب
